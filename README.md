# 清道夫 - Termux空间清理工具

一个专门用于Termux环境的命令行空间清理工具，可以智能识别并清理垃圾文件，释放存储空间。

## 功能特性

- 智能识别多种类型的垃圾文件
- 扫描缓存目录（npm、yarn、系统缓存等）
- 清理前显示详细的文件列表和大小统计
- 每次清理操作都需要用户确认
- 安全的删除机制，避免误删系统文件
- 清晰的中文界面

## 支持的垃圾文件类型

- 日志文件 (`.log`)
- 临时文件 (`.tmp`, `.temp`)
- 编辑器备份文件 (`~`)
- Vim交换文件 (`.swp`)
- macOS系统文件 (`.DS_Store`)
- Windows缩略图缓存 (`Thumbs.db`)

## 安装

```bash
# 克隆或下载项目
cd /root/projects/qingdaofu

# 直接运行
node src/index.js

# 或者使用npm link创建全局命令
npm link
qingdaofu
```

## 使用方法

### 基本用法

```bash
# 清理当前用户目录
node src/index.js

# 清理指定目录
node src/index.js /path/to/directory

# 如果已安装为全局命令
qingdaofu
qingdaofu /path/to/directory
```

### 使用流程

1. 运行扫描命令
2. 工具会扫描并列出所有发现的垃圾文件
3. 显示文件类型、大小和路径
4. 询问是否确认清理
5. 输入 `y` 或 `yes` 确认清理，其他任何输入取消操作

## 安全机制

- 自动跳过系统关键目录（`/system`, `/data`, `/proc`, `/sys`, `/dev`）
- 自动跳过符号链接
- 清理前需要用户确认
- 显示删除失败的文件和错误信息
- 限制扫描深度，避免过度扫描

## 最佳实践

1. 定期清理缓存目录，保持系统整洁
2. 清理前仔细查看文件列表，确认没有误识别
3. 重要文件不要使用垃圾文件扩展名
4. 建议每周清理一次
5. 清理前可以备份重要数据
